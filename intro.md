# Introduction


## The Java Track

This gitbook accompanies several Java-based courses of the bioinformatics curriculum of the Hanze University of Applied Science. These courses include "Introduction Java", "Web-based information systems", "Application Design" and the minor "Application Design".

It covers the basics of Java (Part I - III), some OOP fundamentals (Part IV), Java for the web using Servlets and Thymeleaf (Part V), JUnit testing (Part VI), Advanced design principles including Design Patterns (Part VII), and some tutorials on the Spring framework (Part VIII).
